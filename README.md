# LaboBeton

**LaboBeton** est une application métier dédiée à la gestion et au suivi complet des laboratoires d'essais béton. 

---

## À propos du projet
Le but de **LaboBeton** est de fournir un outil professionnel permettant de piloter l'ensemble de l'activité d'un laboratoire :
* **Suivi de chantiers** : Centralisation des données critiques (clients, entreprises, lieux).
* **Données techniques** : Gestion rigoureuse des éprouvettes béton, dates de réception, de prélèvement et d'écrasement.
* **Mesures terrain** : Saisie du Slump (affaissement), températures et conditions de cure.
* **Automatisation** : Génération instantanée des Procès-Verbaux (PV) associés pour les envoyer aux entreprises avec qui l'utilisateur travaille.

## Project Overview
**LaboBeton** is an app dedicated to managing a concrete testing laboratory. The goal is to provide a business-oriented tool to track different construction sites and their key data:
* **Site Management**: Concrete samples, reception, sampling, and crushing dates.
* **Technical Measurements**: Slump measurements, temperatures, and curing conditions.
* **Reporting**: Automated generation of test reports (PV) to send to the partner companies the user works with.

---

## Fonctionnalités Clés / Key Features

- **Traçabilité totale** : Du prélèvement sur chantier jusqu'à l'écrasement à 28 jours.
- **Calculs Normatifs** : Calcul automatique des masses volumiques et résistances (MPa) selon les normes NF EN 12350 / 12390.
- **Planning Dynamique** : Vue calendrier pour gérer les échéances d'écrasement.
- **Édition PDF** : Rapports personnalisés avec logo et diagnostic de conformité automatique.

## Stack Technique

- **Frontend** : React.js (Vite)
- **Backend** : Node.js & Express
- **Database** : MongoDB (Atlas)
- **Containerization** : Docker
- **Cloud** : Google Cloud Run
